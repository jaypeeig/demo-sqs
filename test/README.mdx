# Button Component

The Button component is a versatile UI element that allows users to trigger actions on the page. It supports multiple styles, sizes, and states to fit different use cases.

## Usage

To use the `Button` component, import it and place it where needed in your React component:

```jsx
import { Button } from 'your-component-library';

<Button label="Click me" />
```
